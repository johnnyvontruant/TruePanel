# Code of Conduct

TruePanel is built through technical curiosity, careful evidence, and respect for the people doing the work.

## Expected behavior

Participants should:

- communicate clearly and professionally
- critique ideas and evidence rather than people
- welcome questions and differing experience levels
- document safety assumptions and uncertainty
- give credit for discoveries, testing, and prior work
- protect private information found in logs or diagnostics
- help keep hardware experiments bounded and reproducible

## Unacceptable behavior

The project does not accept harassment, discrimination, threats, deliberate humiliation, doxxing, credential exposure, destructive hardware advice, or knowingly unsafe instructions.

## Scope

This policy applies to repository discussions, issues, pull requests, reviews, documentation, community spaces, and project-related private communication.

## Enforcement

Project maintainers may edit or remove content, close discussions, reject contributions, or restrict participation when behavior threatens people, project safety, or the integrity of the hardware evidence.

Report conduct concerns privately through the repository maintainers or GitHub reporting tools. Reports should be handled discreetly and without retaliation.
